🚲 Bike Buyers Analysis — Excel Data Analytics Project


📊 Project Overview

This Excel data analysis project explores customer demographics and purchasing behavior to identify the factors influencing bike purchases.

The analysis includes:

- Data cleaning
- Data transformation
- Pivot table analysis
- Interactive dashboard creation
- Insight generation

This project demonstrates practical Excel skills for data analysis, business intelligence, and dashboard design.


🎯 Business Problem

A bike company wants to understand:

- Who buys bikes?
- What income level buys more bikes?
- Does age influence purchase?
- Does commute distance matter?
- Which region buys more bikes?

This project answers these questions using Excel.


📁 Dataset Description

The dataset contains the following fields:

- ID
- Marital Status
- Gender
- Income
- Children
- Education
- Occupation
- Home Owner
- Cars
- Commute Distance
- Region
- Age
- Purchased Bike


🧹 Data Cleaning Process

Data cleaning was performed in WorkingSheet:

✔ Standardized Gender values (M → Male, F → Female)
✔ Standardized Marital Status (M → Married, S → Single)
✔ Created Age Brackets (Young, Middle Age, Old)
✔ Removed duplicates
✔ Formatted Income column
✔ Prepared dataset for Pivot Tables


📈 Analysis Performed

Pivot Tables were used to analyze:

1. Income vs Bike Purchase

- Compared average income of buyers vs non-buyers
- Split by gender

2. Age Bracket vs Purchase

- Young
- Middle Age
- Old

3. Commute Distance vs Purchase

- 0–1 Miles
- 1–2 Miles
- 2–5 Miles
- 5–10 Miles
- 10+ Miles

4. Region vs Bike Purchase

- Europe
- North America
- Pacific

5. Gender vs Purchase

- Male vs Female comparison


📊 Dashboard Features

The Excel dashboard includes:

📌 Bike Purchases by Gender
📌 Bike Purchases by Age Bracket
📌 Bike Purchases by Commute Distance
📌 Bike Purchases by Region
📌 Income Comparison (Buyers vs Non-Buyers)

The dashboard provides interactive visual insights into customer purchasing behavior
